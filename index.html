De LEGO UCS (Ultimate Collector's Series) Venator is een indrukwekkende bouwset van LEGO die is gebaseerd op het iconische Venator-klasse sterrenvernietiger-schip uit het Star Wars-universum. Deze set is ontworpen voor ervaren LEGO-bouwers en biedt een gedetailleerde en nauwkeurige weergave van het schip, met aandacht voor zowel esthetiek als details. Deze set komt rechtstreeks uit de Clone Wars serie. deze set bestaat uit 5374 stukjes, en komt met 2 minifiguren. Een phase two captain Rex en een Clone Wars versie van admiraal Yularen. De set wordt gereleased op 1-10-2023.
